# Si1ence Client

Custom Eaglercraft client combining Tuff Client UT15 base with Galactic Client 6.6.0 modules.

## Overview

Si1ence Client is a hybrid Eaglercraft 1.12 WASM-GC client that merges the robust feature set of **Tuff Client UT15** with the advanced module system from **Galactic Client 6.6.0**.

## Features

### From Tuff Client UT15
- **Fixed "Invalid Payload Capacity" error** on some servers
- **Rendering optimizations** for better performance
- **ViaBlocks integration** with chunk processing queue and y0 queue
- **Fixed ViaBlocks collisions** and rendering
- **WAILA support** for block information
- **TabComplete UI** (disabled by default)
- **W-Tap Trainer, CPVP mode, Shield Status, Better Hitboxes, Fast Crystals, Chat Heads**
- **Sodium and video settings UI**
- **AutoLogin with settings**
- **Elytra fixes**
- **Recording Studio**
- **Customizable Crosshairs**
- **Minimap**
- **Zoom functionality**
- **Full ClickGUI with search and sorting**
- **Entity Culling**
- **WorldEdit CUI**
- **Keystrokes and CPS Counter**
- **Durability GUI**
- **And much more!**

### From Galactic Client 6.6.0
- Additional module integrations
- Enhanced client stability
- WASM-GC version for modern browsers
- Optimized performance tweaks

## Build Information

- **Base**: Tuff Client UT15 WASM (Offline version)
- **Eaglercraft Version**: 1.12 WASM-GC
- **Build Date**: June 2026
- **Compatibility**: Modern browsers with WebGL and WebAssembly support

## Installation

1. Download the `Si1ence_Client_Offline_WASM.html` file from [Releases](https://github.com/ScorchedIV/Si1ence-Client/releases)
2. Open it in your web browser (Chrome, Firefox, Edge, Safari)
3. No installation required - runs entirely in your browser!

## Usage

1. Open the HTML file in your browser
2. Join any Eaglercraft 1.12 compatible server
3. Access modules via the ClickGUI (default keybind)
4. Customize settings in the Video Settings menu
