# Si1ence Client - Available Modules

## Core Modules

### Client Utilities
- **ClickGUI** - Main menu for accessing modules
- **Tab Complete** - Server command auto-completion (disabled by default)
- **Minimap** - Small map showing your surroundings
- **Zoom** - Scroll wheel zoom functionality
- **Recording Studio** - Built-in recording features

### Visual Enhancements
- **Fullbright** - Remove darkness from dark areas
- **Custom Crosshair** - Customize your crosshair appearance
- **Keystrokes Display** - Show keys you're pressing
- **CPS Counter** - Display clicks per second
- **Durability GUI** - Show item durability
- **Chat Heads** - Display player avatars above chat messages

### Gameplay Features
- **Quick Elytra** - Fast elytra deployment
- **AutoLogin** - Automatically login with saved credentials
- **NoRain** - Remove rain from the world
- **NoGlint** - Disable item enchantment glint
- **Crouching Under Blocks** - Automatically crouch when needed
- **Fast Crystals** - Faster end crystal placement
- **Better Hitboxes** - Improved entity hitbox display
- **CPVP Mode** - Combat-focused settings
- **W-Tap Trainer** - Practice W-tapping technique
- **Shield Status** - Show shield durability/status

### Server Features
- **Waypoints** - Create and save waypoints
- **AntiPickup** - Prevent automatic item pickup
- **Y0 Support** - Support for TuffX plugin servers
- **Wisp Support** - Compatible with Wisp servers
- **Server List** - Save favorite servers

### Visual Overlays
- **WAILA** - What Am I Looking At (block info)
- **WorldEdit CUI** - WorldEdit selection display
- **Glowing Ores** - Highlight valuable ores
- **TNT Countdown** - Show TNT explosion timer

### Advanced Rendering
- **ViaBlocks Support** - 1.13+ block rendering on 1.12
- **ViaItems Support** - 1.13+ item rendering on 1.12
- **Bubble Column Support** - Proper bubble column physics
- **Entity Culling** - Improved entity rendering
- **Sodium Video Settings** - Advanced graphics options

### Experimental Features
- **Shaders** (Experimental) - Shader support (may not be stable)
- **Tuff Mods** - Tuff Network custom modifications

## Module Settings

### How to Access Module Settings

1. Open ClickGUI
2. Find the module in the list
3. Click the module to toggle it on/off
4. Right-click or click settings icon for advanced options

### Common Settings

- **Enabled/Disabled** - Toggle module on or off
- **Key Binding** - Set hotkey to toggle module
- **Color** - Customize color of display
- **Scale** - Adjust size of displays
- **Position** - Move displays around screen
- **Transparency** - Adjust opacity

## Troubleshooting Modules

### Module Won't Enable
- Check server compatibility
- Ensure you're on an Eaglercraft 1.12 server
- Try disabling conflicting modules
- Restart the client

### Module Causes Lag
- Reduce scale/quality settings
- Disable if not needed
- Check for conflicts with other modules

### Missing Module Options
- Some modules have limited customization
- Check if module requires specific server features

## Performance Tips

1. **Disable unused modules** - Each active module uses resources
2. **Lower video settings** - Use Sodium settings to optimize
3. **Reduce render distance** - Helps with FPS
4. **Close other tabs** - Frees up memory
5. **Use latest browser version** - Better performance
